# Nabadhara E-commerce 🛒

A full-featured e-commerce web application built with **Laravel**, **Blade**, **PHP**, **CSS**, and **JavaScript**.  
This project provides a complete online shopping solution with product management, customer accounts, and order handling. It is responsive, scalable, and ready for deployment.

---

## 📂 Repository
GitHub Repository: [nabadhara-ecommerce](https://github.com/Md-Moklesar-Rahman-Bappy/nabadhara-ecommerce)

---

## ✨ Features
- 🛍️ Product catalog with categories  
- 👤 Customer registration and login system  
- 🛒 Shopping cart and checkout process  
- 📦 Order management for admins  
- 📱 Responsive design for desktop and mobile  
- 🔐 Secure authentication and storage  
- 🌐 SEO-friendly structure with `sitemap.xml`

## 📦 Installation & Setup
To run this project locally:

1. **Clone the repository:**
```bash
   git clone https://github.com/Md-Moklesar-Rahman-Bappy/nabadhara-ecommerce.git
```
2. **Navigate into the project folder:**
```bash
   cd nabadhara-ecommerce
```
3. **Install dependencies:**
```bash
   composer install
    npm install
```
4. **Set up environment variables:**
• 	Copy .env.example to .env
• 	Configure database connection (MySQL recommended)
5. **Run migrations and seed database:**
```bash
   php artisan migrate --seed
```
6. 	Compile assets:
```bash
   npm run dev
```
7. 	Start local server:
```bash
   php artisan serve
```
8. 	Open in browser:
```bash
   http://127.0.0.1:8000
```

## 🌍 Deployment
1. Deployable on shared hosting, VPS, or cloud platforms (e.g., AWS, DigitalOcean).
2. Requires PHP 8+, MySQL, and Composer.
3. Configure .env for production database and mail settings.
4. Use php artisan config:cache and php artisan route:cache and for optimization.
